## How to Use

1. Browser opened to http://127.0.0.1:8000
   If not, manually open: http://127.0.0.1:8000

2. Drop a WAV/FLAC/AIFF file in the drop zone

3. Configure pipeline settings (left panel):
   - Target loudness: -23 LUFS (EBU), -27 (Netflix), -14 (Streaming)
   - Stem model: htdemucs_6s, htdemucs_ft, spleeter:5stems
   - Silence gate: -50, -48, -45 dBFS
   - Output format: WAV, AC-3, DTS

4. Adjust studio tuning if needed (⚙ button top-right):
   - Genre presets: Pop, Rock, Electronic, Jazz, Hip-Hop, Cinematic
   - Tape saturation, harmonic drive, console model
   - Bus compression, transient punch, parallel crush
   - EQ: low-end punch, mid presence, air/sparkle
   - 5.1 spatial: width, rear depth, room reverb, LFE crossover

5. Click "▶ RUN FULL PIPELINE"

6. Wait for all 7 stages to complete (real processing takes time)

7. Click "⬇ EXPORT FILES" to download:
   - 6-channel WAV (24-bit)
   - Dolby AC-3 (640 kbps)
   - DTS (1509 kbps)
