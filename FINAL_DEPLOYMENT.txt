## How to Use

1. Open: http://127.0.0.1:8000

2. Drop audio file (WAV/FLAC/AIFF) - up to 500 MB

3. Configure:
   - Target loudness: -23 LUFS (EBU), -27 (Netflix), -14 (Streaming)
   - Stem model: htdemucs_6s, htdemucs_ft, spleeter:5stems
   - Silence gate: -50, -48, -45 dBFS

4. Studio Tuning (⚙ button):
   - Genre presets: Pop, Rock, Electronic, Jazz, Hip-Hop, Cinematic
   - Analog color: Tape, harmonic drive, console
   - Dynamics: Bus comp, transient, parallel
   - EQ: Low, mid, air
   - 5.1 Spatial: Width, rear, reverb, LFE

5. Run Pipeline

6. Export: WAV, AC-3, DTS
