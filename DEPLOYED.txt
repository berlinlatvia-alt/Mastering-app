## How to Use

1. Browser should auto-open to http://127.0.0.1:8000
   If not, manually open: http://127.0.0.1:8000

2. Drop a WAV/FLAC/AIFF file in the drop zone

3. Configure pipeline settings (left panel)

4. Adjust studio tuning if needed (⚙ button top-right)

5. Click "▶ RUN FULL PIPELINE"

6. Wait for all 7 stages to complete

7. Click "⬇ EXPORT FILES" to download

## To Stop Server

- Close the terminal window, OR
- Press Ctrl+C in the terminal, OR
- Run: taskkill /F /IM python.exe
